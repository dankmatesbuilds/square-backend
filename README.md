# square-backend

Minimal example backend for Square (Node.js + Express + Square SDK).

Quick start

1. Copy `.env.example` to `.env` and set your Square credentials:

   SQUARE_ACCESS_TOKEN=YOUR_ACCESS_TOKEN
   SQUARE_LOCATION_ID=YOUR_LOCATION_ID
   SQUARE_ENV=sandbox

2. Install dependencies and start the dev server:

   npm install
   npx vercel dev

3. Health check:

   GET http://localhost:3000/api/health

4. Create a payment (example using sandbox test nonce):

   POST http://localhost:3000/api/checkout
   Content-Type: application/json

   {
     "sourceId": "cnon:card-nonce-ok",
     "amount": 100,
     "currency": "USD"
   }

Notes
- This is a Vercel serverless project. Each file in `api/` is a separate endpoint.
- This server expects a card nonce/token (`sourceId`) generated by Square Web Payments SDK on the frontend or you can use the sandbox special nonces for testing.
- Do not commit your real access token. Use environment variables or a secrets manager in production.
